# 台股模擬交易系統

## 系統說明

這是一套每日自動分析台股、產生買賣訊號、模擬交易並計算盈虧的系統。
初始本金 **20 萬元**，使用多重技術指標策略自動決策。

---

## 安裝步驟（只要做一次）

### 1. 確認有安裝 Python 3.9 以上

```bash
python --version
```

### 2. 安裝套件

```bash
pip install -r requirements.txt
```

---

## 每天怎麼用

### 方法一：手動跑（收盤後執行）

```bash
python run_daily.py
```

執行後會：
1. 自動抓台股資料
2. 分析訊號
3. 執行虛擬買賣
4. 在 `reports/latest_report.json` 產生報告

然後用瀏覽器開啟 `dashboard.html` 看結果。

---

### 方法二：設定排程自動執行（推薦）

#### Windows（工作排程器）
1. 搜尋「工作排程器」並開啟
2. 建立基本工作
3. 觸發程序選「每天」，時間設 **15:00**（收盤後）
4. 動作選「啟動程式」，程式路徑選你的 `python.exe`
5. 引數輸入 `run_daily.py`，起始位置填本資料夾路徑

#### Mac / Linux（crontab）
```bash
crontab -e
# 加入以下一行（每天下午 15:05 執行）
5 15 * * 1-5 cd /你的路徑/taiwan_stock_sim && python run_daily.py
```

## 看報告

用瀏覽器直接開啟 `dashboard.html`，就能看到：

- 📊 帳戶總覽（總資產、損益、現金）
- 📈 累計損益曲線
- 🔴🟢 今日交易記錄（買了什麼、賣了什麼、幾錢）

- 💡 全部候選股今日訊號

## 策略說明

系統同時使用以下 4 種技術指標，綜合判斷：

| 指標 | 買入條件 | 賣出條件 |
|------|---------|---------|
| 均線交叉 | MA5 上穿 MA20（黃金交叉）| MA5 下穿 MA20（死亡交叉）|
| RSI | RSI < 30（超賣）| RSI > 70（超買）|
| 布林通道 | 股價觸碰下軌 | 股價觸碰上軌 |
| 成交量 | 配合以上訊號放量確認 | 配合以上訊號放量確認 |

- 停損：持倉虧損超過 **-8%** 自動賣出
- 停利：持倉獲利超過 **+15%** 自動賣出
- 同時最多持有 **5** 支股票
- 每次買入約 **3 萬元**

## 候選股清單

目前追蹤台灣 50 成份股中的 25 支大型股，涵蓋：
- 科技：台積電、鴻海、聯發科、廣達、台達電、華碩
- 金融：富邦金、國泰金、兆豐金、玉山金、中信金
- 傳產：台塑、南亞、台塑化、中鋼
- 航運：長榮、陽明、萬海、華航
- 電信：中華電

你可以修改 `run_daily.py` 裡的 `CANDIDATE_STOCKS` 清單，加入你有興趣的股票代碼。

## 注意事項

任何技術指標策略都無法保證獲利，真實交易請自行判斷。
